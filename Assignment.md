# 📌 Assignment Submission Instructions (CSE-1202)

## Assignment Title
**Creating Custom Header Files in C Programming**

---

## 🎯 Objective
In this assignment, you will learn how to:

- Create custom header files in C
- Organize reusable functions
- Include header files in another C program
- Upload and submit your code using GitHub

---

## 📋 Task Requirements

You have to create a custom header file named:

```c
abc.h
```

This header file should contain implementations of the following basic functions:

### Required Functions

- `max()` → Find maximum of two numbers
- `min()` → Find minimum of two numbers
- `isPrime()` → Check whether a number is prime
- `isOdd()` → Check whether a number is odd
- `isEven()` → Check whether a number is even
- `digitSum()` → Calculate sum of digits
- `reverseNumber()` → Reverse a number

### Number Conversion Functions

Implement at least some number conversion functions such as:

- Decimal to Binary
- Binary to Decimal
- Decimal to Octal
- Decimal to Hexadecimal

### Additional Functions

You are encouraged to add more useful basic functions, such as:

- `factorial()`
- `gcd()`
- `lcm()`
- `power()`
- `palindrome()`

---

## Main Program

Create another C file named:

```c
test.c
```

In this file, you have to:

- Include your header file
- Call and test all functions

Example:

```c
#include "abc.h"

int main() {
    // Test your functions here
    return 0;
}
```

---

## Folder Structure

Your project should look like this:

```bash
assignment(CSE-1202)/
│
├── abc.h
├── test.c
└── README.md   (optional)
```
